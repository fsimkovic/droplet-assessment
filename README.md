# Droplet size analysis tool 

## Installation

```bash
$ pip install -r requirements.txt
```

## Execution

```bash
$ ./main data/videos/1-microlitre.mp4
```

```bash
$ ./main --debug data/videos/1-microlitre.mp4
```

```bash
$ ./main --csv data/videos/1-microlitre.mp4 data/videos/2-microlitre.mp4
```
